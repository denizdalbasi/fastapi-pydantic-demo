# Pydantic & FastAPI Example Project

This project demonstrates how to use **Pydantic** for data validation and **FastAPI** for building simple REST APIs.

---

##  Features

- User data validation with custom validators
- Product model with optional fields
- FastAPI endpoints for users and search
- Clean and well-documented code

---

## Technologies Used

- Python 3.9+
- Pydantic
- FastAPI
- Uvicorn (for running the server)
